# gyyanX Website

This is the static marketing website for **gyyanX – Intelligence That Transforms**.

## Structure

- `index.html` — main single-page site.
- `articles/rl-gnn-query-optimization.html` — example article / technical brief.

## How to use with GitHub Pages

1. Create a new public repo on GitHub, e.g. `gyyanx-site`.
2. Upload `index.html`, the `articles/` folder, and this `README.md`.
3. In the repo, go to **Settings → Pages**.
4. Under **Source**, choose `Deploy from a branch`, branch `main`, folder `/ (root)`.
5. Save. GitHub Pages will give you a public URL.
6. Point your GoDaddy domain (gyyanx.com) to this GitHub Pages site using DNS (CNAME/A records).
